# Password-Analyser

This project is a cybersecurity enthusiast project. Every cyber learner builds these. I built this as my fully own project.

## What's implemented inside this?

- We have a score generator, that generates score according to the length of your password, diversity between the cases, special characters, and digits.

- We have input for password input by user. 

- We have output for printing the score, and telling basic info, like based on score, your password is - weak, moderate, or strong.

#### That's it what we have till now. Open for more features.
